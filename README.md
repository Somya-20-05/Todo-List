# to-do List

A simple to-do list project built by HTML,CSS,JAVASCRIPT

## features
- first you add the task
- click the add button to add in the list
- the list has checkbox
- when you check the checbox ,the task is added to a separated list of completed task today

## technologies Used 

- HTML
- CSS
- Basic JAVASCRIPT (Event Handling)
